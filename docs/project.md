# Project overiew

## Product Name
Car Marketplace Platform

## Description
This is a web application where users can:
- Buy cars
- Sell cars
- Browse listings

## Key Features
- User authentication
- Car listing management
- Search and filter cars
- Order/booking system

## User Roles
- Buyer
- Seller
- Admin

## Core Modules

### 1. Authentication
- Register
- Login
- JWT authentication

### 2. Car Listings
- Create listing
- Update listing
- Delete listing
- View listings

### 3. Orders
- Book car
- View order history

## Business Rules
- Only sellers can create listings
- Buyers can only view and purchase
- Listings must have price and details
-Admin can see ued and lock/unlock them as well as theire listings too

## Tech Stack
- Backend: .NET Web API
- Frontend: Angular
- Database: SQL Server
